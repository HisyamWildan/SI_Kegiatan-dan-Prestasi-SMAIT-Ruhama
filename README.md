# 🏫 Sistem Informasi Kegiatan dan Prestasi Sekolah

**Sistem Informasi Kegiatan dan Prestasi Sekolah** adalah aplikasi berbasis web yang dirancang untuk membantu pengelolaan, dokumentasi, dan publikasi kegiatan serta prestasi siswa di lingkungan **SMAIT Ruhama Depok**. Sistem ini dikembangkan sebagai bagian dari proyek Skripsi Berdampak Program Studi Teknik Informatika STT Terpadu Nurul Fikri.

---

## 🎯 Tujuan Proyek

* Menyediakan platform terpusat untuk pengelolaan data kegiatan dan prestasi sekolah.
* Memudahkan guru dan pihak sekolah dalam mendokumentasikan kegiatan serta prestasi siswa.
* Membantu proses publikasi informasi kegiatan dan prestasi kepada siswa, orang tua, dan masyarakat.
* Mengurangi risiko kehilangan data melalui penyimpanan digital yang terstruktur.
* Mendukung transformasi digital administrasi sekolah yang lebih efektif dan efisien.

## 📈 Metode Pengembangan

Sistem dikembangkan menggunakan metode **Extreme Programming (XP)** yang terdiri dari tahapan:

* Planning
* Design
* Coding
* Testing

Metode ini dipilih karena mendukung pengembangan yang fleksibel, iteratif, dan mampu beradaptasi terhadap perubahan kebutuhan pengguna selama proses pembangunan sistem.

---

## 🧪 Pengujian Sistem

Pengujian sistem dilakukan menggunakan:

* ✅ Black Box Testing untuk menguji fungsionalitas fitur
* ✅ User Acceptance Testing (UAT) untuk mengukur tingkat penerimaan pengguna

Hasil pengujian menunjukkan bahwa sistem dapat berjalan sesuai kebutuhan pengguna dan memperoleh tingkat penerimaan yang sangat baik dari pihak sekolah.
